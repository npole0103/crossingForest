## 숲속을 걸어요
```
모바일 게임 '길건너 친구들'을 모티브로 제작한 2인 게임입니다.
게임을 플레이 할 뿐만 아니라 서버와 클라이언트간의 의사소통 기능이 제공됩니다.
```
## 게임 플레이 방법
1. mine\Build 폴더 내에 mine 실행파일 실행

## 게임 조작법
>1P
>>이동 : W,S,A,D
>>
>>점프 : 스페이스바

>2P
>>이동 : 방향키
>>
>>점프 : 넘버패드 0

## 서버와 클라이언트 연결
1. VisualStudio\CsConnect\bin\Debug 파일 내에 실행파일 실행
2. 게임 실행
3. 연결에 성공하면 서버와 클라이언트가 메세지를 주고받을 수 있음
